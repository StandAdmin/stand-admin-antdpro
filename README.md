# CRUD framework works perfectly with Antd Pro

### Important

Disable Dva model validate in [config/config.ts](https://github.com/rooseve/stand-admin-antdpro-demo/blob/main/config/config.ts), like this:

```
dva: {
   skipModelValidate: true,
},
```
